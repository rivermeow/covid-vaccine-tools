Tools to assist in finding and booking vaccine appointments. More tools coming soon.

Rite Aid form filler

Fills the form with your data and navigates to the store list page

Usage:

  1. Edit the bookmarklet with your values as needed, create a new bookmark, and copy your version to the address field
  2. See below for mobile browser instructions
  3. Go to the Rite Aid Covid Qualifier and click the bookmarklet
  4. Your browser will complete the form and navigate to the store selection page.

Rite Aid status bookmarklet

Adds a green checkbox or red X next to each store indicating whether the store MAY have vaccines or if none are available and a slot count per day.

Update as of 3/16 Now the number of slots available and slot count per day will show for each store with availability. If you previously added the bookmarklet to your browser, please add it again using the instructions below.

Usage:

  1. On a desktop browser, install the bookmarklet in your browser by dragging this link: Rite Aid Status Check to your bookmarket bar.
  2. See below for mobile browser instructions
  3. Go to the Rite Aid Covid Qualifier and complete the form (or use the form filler bookmarklet above)
  4. On the results page, click the bookmarklet to see the status next to each store
  5. Choose a store with availability and schedule your appointment.

Note that availability is not always correct and stores that indicate availability may not actually have availability
